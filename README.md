# Advent Of Code 2021

Doing the Advent of Code 2021 challenges in Haskell.

## Usage

### Stack

```shell
> stack build
> stack run day-01
> stack test :test-01
```

```shell
> stack repl adventofcode2021:lib
...
>>> :load Day01
```

### Cabal

```shell
> cabal build
> cabal run day-01
> cabal run test-01
```

```shell
> cabal repl adventofcode2021
...
>>> :load Day01
```
